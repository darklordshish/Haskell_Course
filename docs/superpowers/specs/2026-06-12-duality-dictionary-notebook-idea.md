# Идея (черновик): ноутбук-словарь двойственных конструкций

**Дата:** 2026-06-12. **Статус:** ЧЕРНОВИК-ЗАМЕТКА (не спека к исполнению). Полноценный
брейншторм + спека — когда пользователь скажет «строим».

## Замысел

Ноутбук-справочник — «парад двойственных категорных конструкций». Пользователь будет
периодически дополнять. Формат: сводная таблица-оглавление (кратко, с мини-диаграммами) →
по ссылкам подробные разделы (матопределение + универсальное свойство + Haskell-реализация
обоих членов пары).

## Предлагаемая структура (обсудить при брейншторме)

- **Сводная таблица** в начале: `Конструкция | Двойственная | Идея | мини-диаграмма |
  Haskell | →подробнее`.
- **Раздел на пару**: определение, универсальное свойство, диаграмма в стиле курса
  (обращение стрелок = двойственное), Haskell обоих.
- Кандидаты-пары: product/coproduct, terminal/initial, equalizer/coequalizer,
  pullback/pushout, limit/colimit, mono/epi, algebra/coalgebra, monad/comonad,
  cata/ana (fold/unfold), Lan/Ran, free/cofree, adjunction left/right.

## Открытые вопросы (для брейншторма)

- Размещение: отдельный reference-ноутбук вне ствола 0–VI (приложение/словарь)?
  Имя файла (напр. `Duality.ipynb`), и куда ссылаться из тем.
- Оглавление: вручную или генерируемое из разделов (как `deps_header`/`course_map`)?
- Объём начального набора пар vs «дописывать со временем».
- Использует кит стрелок (mono ↣ / epi ↠ / iso ↔ / nat ⇒) и светлую палитру из
  Faza 6 / Фазы B — диаграммы рисуются по этому канону (см. DesignShowcase).
